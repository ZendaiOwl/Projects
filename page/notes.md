# Soldering temperatures

| Soldering Temperature Chart |  |  |
| :-- | :-- | :-- |
| Solder Type | Lead / Non-Lead | Temperature (°C) |
| 63/37 | lead | 183 |
| 60/40 | lead | 183-188 |
| 50/50 | lead | 183-212 |
| 45/55 | lead | 183-224 |
| 40/60 | lead | 183-234 |
| 96S | lead | 221 |
| 95A | lead | 236-243 |
| Alloy No. 1 | lead | 183-215 |
| Alloy No. 2 | lead | 183-190 |
| HMP 5S | lead | 296-301 |
| LMP 62S | lead | 179 |
| TLS/5 | lead | 296-301 |
| TIN | tin | 232 |
| 99C | non-lead | 227 |
| 97C | non-lead | 230-250 |
| SAC3 | non-lead | 217-219 |
| MC1 | non-lead | 232 |
